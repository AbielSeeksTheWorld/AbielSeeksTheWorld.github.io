Install Abiel

This is a lightweight app you can add to your home screen.

iPhone / iPad (Safari)

  1. Open this site in Safari
  2. Tap Share (square with arrow)
  3. Tap Add to Home Screen

Android (Chrome)

  1. Open this site in Chrome
  2. Tap the menu ⋮
  3. Tap Install app (or Add to Home screen)

Tip: If you see an install icon in the address bar, tap it.
